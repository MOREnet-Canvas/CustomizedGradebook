Parent Mastery Dashboard Setup Instructions

For Canvas Admins

This guide will help you enable the Parent Mastery Dashboard for courses in the Canvas Parent app. Parents/observers will see a button on the course Front Page that links to a mastery tracking dashboard.

⚠️ Prerequisite: Install Mobile Loader

Before setting up individual courses, you must install the Mobile Loader once in your Canvas Mobile Theme.

The loader provides the JavaScript code that powers the Parent Mastery Dashboard. Without it, the dashboard will not function.

Installation:

  1. Go to Account → Themes → Mobile
  2. Click Edit on your mobile theme
  3. Download the loader: mobile_loader.js
  4. Paste the entire file into the JavaScript section
  5. Click Save

Version selection: Edit the MOBILE_VERSION variable in the loader:

  • "mobile-dev" - Latest development build (auto-updated)
  • "mobile-v0.1.1" - Specific stable release (recommended for production)

📖 Detailed installation guide

Per-Course Setup

After installing the mobile loader, follow these steps for each course where you want to enable the Parent Mastery Dashboard:

Step 1 — Get the Course ID

Open the course and look at the URL:

/courses/38297

The number after /courses/ is the Course ID.

Step 2 — Create the Mastery Dashboard Page

  1. Go to Pages.
  2. Click + Page.
  3. Title it: Mastery Dashboard
  4. In HTML Editor paste:
<div id="mastery-dashboard-root"></div>
  1. Click Save & Publish.

Step 3 — Add Button to Front Page (Optional)

Note: If you installed the mobile loader, the button may appear automatically. If you prefer to add a custom button or the automatic button is not appearing, follow these steps:

Enter the Course ID and copy the generated button code below.

Step 4 — Paste Into Front Page

  1. Go to Pages → Front Page.
  2. Click Edit.
  3. Switch to HTML Editor.
  4. Paste the generated code at the very top.
  5. Click Save & Publish.

Important Notes

  • The mobile loader must be installed for the dashboard to work.
  • The ?cg_web=1 parameter is required for the Parent app webview.
  • All links are relative, so this works in test, beta, and production environments.
  • The dashboard only renders data for observers (parent accounts).
  • Each course needs its own "Mastery Dashboard" page created (Step 2).
