# Admin Dashboard

The Customized Gradebook Admin Dashboard provides configuration options for Canvas administrators.

## Overview

The Admin Dashboard allows you to:

- Configure account filtering
- Manage feature toggles
- View system status
- Access configuration tools

## Accessing the Dashboard

1. Log in to Canvas as an admin
2. Navigate to the root account
3. Look for the Customized Gradebook admin panel

## Features

### Account Filtering

Control which sub-accounts have access to Customized Gradebook features.

See [Account Filtering](account-filtering.md) for details.

### Feature Toggles

Enable or disable specific CG features:

- Student dashboard customizations
- Gradebook enhancements
- Mobile module integration

### Configuration

Manage global settings for the Customized Gradebook.
